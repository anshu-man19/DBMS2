# User Guide

## Company Users

- Register from `/register`.
- Sign in from `/login`.
- Use `/dashboard` to view the recruiter workspace.
- Create JNFs and INFs from the new form pages.
- Review status updates and notifications from the dashboard pages.

## Admin Users

- Sign in from `/admin/login`.
- Use `/admin/dashboard` to monitor the portal.
- Review submitted JNFs and INFs from the queue pages.
- Open company profiles to update recruiter details.

## Status Workflow

- `draft` for in-progress forms.
- `submitted` after company submission.
- `under_review` when admin starts review.
- `accepted` when approved.
- `rejected` when declined.

## Notifications

- Company users receive updates when submissions change status.
- Admins receive notifications for new company registrations and new submissions.
